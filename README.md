# Explorer

Arrow keys to move
Space or Esc to escape entrapment

Clear all the tiles by walking on them.
 - Red tiles need 1 step
 - Orange tiles need 2 steps
 - Green tiles need 3 steps
 - ...
 - Black tiles need an undeterminably large number of steps
